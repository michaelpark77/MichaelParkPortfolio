Project 8

Nintendo Price

Nintendo Price 소개
Nintendo Price 는 사이드로 진행한 프로젝트로 각국의 온라인 게임 판매 스토어인 eshop 내의 닌텐도 스위치의 게임 판매 가격을 비교해주는 사이트입니다.
Avedion
Backend : Nodejs, Express, ubuntu
Data : Mongodb, Mongoose, Redis
Frontend : javascript, jquery
닌텐도 스위치 게임들의 경우 각국마다 가격이 다르나 어디서 구매하든 동일한 게임을 받게 됩니다.
매일 환율과 각국(34개국)의 eshop API를 조회해 오고 가격정보를 저장합니다.

Mongodb를 이용해서 게임 정보를 저장하도록 하였고
Redis는 환율 정보를 저장과 캐싱을 하는 용도로 사용하였습니다.
사이트 전체를 반응형으로 제작하였습니다

Main
세일 정보등과 더불어 여러가지 정보를 한눈에 볼 수 있는 랜딩페이지
각종 리스트
일정 기준을 바탕으로 각종 리스트를 바탕으로 각종 리스트를 보여주는 페이지.
세일, 메타크리틱 스코어(게임 전문 사이트), 신작게임 등을
일정한 기준으로 나눠서 보여주는 페이지입니다.
게임 세부 정보 페이지
게임의 가격 정보 등을 보여주는 페이지입니다.
가격의 변동 추이, 각국의 가격비교, 게임 일반 정보를 보여줍니다.
