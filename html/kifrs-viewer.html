Kifrs Viewer

Kifrs Viewer 소개
KIFRS는 한국채택 국제회계기준의 약자입니다.

KIFRS는 기준서임에도 불구하고 검색이나 전체적인 책갈피와 같은 기능이 없어서
검색을 하는 기능과 책갈피 기능이 구현된 편리한 사이트입니다.


Kifrs Viewer
Backend : Node JS, Express, Mysql, Linux
Frontend : Javascript, CSS, jQuery

간단한 프로젝트였기 때문에 간단한 DB설계와 더불어 시도해 보고 싶었던 Node JS를 사용해서 서버를 구축하였고

기존에 사용했던 AWS와 같은 클라우드를 사용하지 않고 리눅스 서버를 구축하여 도메인에 연결시켰습니다.

Index 페이지
처음 들어오게 되면 나오는 페이지로 초기 주제에 대한 검색을 통해서
어떤 기준서에 자신이 찾는 주제가 있는지에 대해서 검색할 수 있도록 해놓았습니다.
기준서 내용 페이지
기준서의 내용을 목차 별로 볼 수 있도록 만들어놓았습니다. 문단 내용에 들어있는 책갈피 링크를 클릭하게 되면 하단에 책갈피 쪽에 추가가 되고 기존의 다른 기준서 사이트와 달리 다른 기준서 페이지로 이동하지 않아도 바로 확인할 수 있습니다.
